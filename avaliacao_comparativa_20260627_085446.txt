Ele usa um controle de congestionamento para lidar com drops de pacotes. UDP (User Datagram Protocol) não garanta a confiabilidade: os dados são enviados apenas o quanto for possível no tempo real, sem garantia de entrega, perdida ou retraso.
Fine-Tuned: TCP (Transmission Control Protocol) oferece confiabilidade: ele garante que pacotes sejam entregues corretamente e de forma sequencial, com controle de erros e retransmissão automatica. UDP (User Datagram Protocol) é mais simples: ele não garanta entrega, sequência ou confirmação, focando em velocidade e eficiência, permitindo cabeçalhos reduzidos.

Fine-Tuned: O operador '&' (endereço de memória) é usado para obter o endereço de um dado na memória. Por exemplo: int x = 5; int* p = &x; printf("%d", *p); // Imprime 5

O operador '*' (desreferenciamento) é usado para acessar o valor armazenado no endereço do ponteiro. Por exemplo: int x = 5; int* p = &x; printf("%d", *p); // Imprime 5 printf("%d", *p++); // Incrementa p e imprime o novo valor, que é 6

[Questão 5]
Pergunta: Qual estrutura de dados utiliza a política LIFO e quais são suas operações principais?
Resposta Esperada: A pilha utiliza LIFO (Last In First Out). Suas operações principais são push para inserir e pop para remover elementos.
Base Model: A Estrutura de Dados com Política FIFO (First-In-First-Out) é uma fila (queue). Ela possui as seguintes operações principais:

1. enqueue(x): Adicionar um elemento x no final da fila.
2. dequeue(): Remover o elemento que está na frente da fila e retorná-lo. Se a fila estiver vazia, retorna None ou um valor padrão.
3. front(): Retorna o elemento atualmente na frente da fila sem removê-lo.
4. size(): Retorna o número de elementos na fila.

Alternativa B
Fine-Tuned: A política LIFO (Last In, First Out) é utilizada por estruturas como Pilha (`Stack`). As operações principais para uma pilha incluem: `push(item)` adiciona um item no topo da pilha; `pop()` remove o item do topo da pilha; `top()` retorna o item do topo sem removê-lo; e `isEmpty()`, que verifica se a pilha está vazia.
